DESIGN SUMMARY - KEY VALUES FOR RECREATION
===============================================================

COLOR PALETTE:
  Primary Background:  #000000 (pure black)
  Primary Text:        #FFFFFF (white)
  Accent Red:          #F74646 (bright red - hover states, links)
  Button Red:          #A22020 (dark crimson - CTA buttons)
  Dark Wine Red:       #7E0F0F (The Furst buttons)
  Deep Red:            #930303 (alternate buttons)
  Medium Crimson:      #962525 (alternate buttons)
  Border Red:          #C30000 (image borders, mobile toggle)
  Light Gray BG:       #F3F5F8 (testimonial/review sections)
  Near White:          #F9FAFD
  Light Blue-Gray:     #C8D5DC
  Page Transition:     #FFBC7D (warm orange)
  Dark Overlay:        #000000CC (80% black)
  Button Hover BG:     transparent (#02010100)

FONTS:
  EVERYTHING uses "Poppins" (Google Font)
  Weights used: 300 (light body), 400 (regular/nav), 600 (semibold headings)

BUTTON STYLE (PRIMARY CTA):
  background: #A22020
  color: #FFFFFF
  border: 1px solid #FFFFFF
  border-radius: 100px (pill)
  padding: 16px 35px (standard) or 16px 60px (wide)
  font: Poppins 16px, 400 weight, capitalize
  hover: color #F74646, bg transparent

IMAGE STYLE:
  Product circles: 5px solid #C30000 border, 100px radius, drop shadow
  Gallery items: 8px border-radius (small) or 25-31px (large)
  Gallery gaps: 15-30px (tight) or 96-100px (spacious)

SECTION DIVIDERS:
  3.7px solid white line, 82% width, centered

LAYOUT:
  Max container: 1140px
  Section padding: typically 8% horizontal
  Widget spacing: 20px
  Sections often overlap with negative margins (-50px)
